# 🍋 Экзотические фрукты

Тренировочный проект по курсу **«Базовый JS. Циклы и массивы»**.

Приложение отображает список экзотических фруктов и позволяет взаимодействовать с ним через панель управления.

## Функциональность

- **Отображение** — список фруктов динамически формируется из данных в формате JSON
- **Перемешивание** — случайное перемешивание карточек; предупреждение если порядок не изменился
- **Фильтрация** — фильтрация по полю `weight` в заданном диапазоне (включая концы)
- **Сортировка** — сортировка по полю `color` с замером времени выполнения
- **Смена алгоритма** — переключение между `bubbleSort` и `quickSort`
- **Добавление** — добавление нового фрукта через форму с валидацией полей

## Стек

## Что отработано

- Работа с массивами: циклы, `filter`, `splice`, деструктуризация
- Генераторы (`function*`, `yield`)
- Работа с DOM: `createElement`, `appendChild`, `innerHTML`
- Алгоритмы сортировки: пузырьковая и быстрая
- Валидация: правила, сообщения, контроллеры

## Запуск

Открыть `index.html` в браузере — сборка не требуется.
